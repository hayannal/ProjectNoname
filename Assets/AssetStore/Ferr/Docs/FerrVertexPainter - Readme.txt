Shortcuts:
---------------------------------
p     : toggle paint mode
c     : quick color select
[ & ] : brush size
; & ' : brush strength
< & > : brush hardness
1 - 4 : quick channel select

Ctrl  + Scroll         : brush size
Shift + Scroll         : brush strength
Ctrl  + Shift + Scroll : brush hardness
